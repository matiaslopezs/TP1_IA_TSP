## Pasos para instalar
- Tener instalado Node JS y NPM
- Instalar quasar cli con ``sudo npm install -g @quasar/cli``
- Estando en la carpeta principal Instalar dependencias con ``npm install``
- Ejecutar el proyecto con ``quasar dev``
